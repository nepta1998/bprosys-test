## Dependencies
- PostgreSQL 14.9
### Instalación de dependencias:

1. Crear un enviroment
    ```
    python3 -m venv env
    ```
2. Activamos es entorno virtual
    ```
    source venv/bin/activate
    ```
3. Instalamos dependencias
    ```
    pip install -r requirements.text
    ```
### Ejecutar script
```
  python respuestas/pregunta56.py
````
